
# Simple Album 

A simple album application made using Angular and NodeJS


## Run Locally

Clone the project

```bash
  git clone https://github.com/suyash-thakur/album
```

For backend go to the backend directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node server
```

  For backend go to the album directory

```bash
  cd album
```
Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve
```
